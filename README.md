# CABATA1 - Verificador de IP 🔍

App simples que mostra informações sobre seu endereço IP.

## Como usar

1. Abra o `index.html` no navegador
2. Clique em "Verificar Meu IP"
3. Veja sua localização, ISP, fuso horário e mais

## Tecnologias

- HTML5
- CSS3 (glassmorphism, gradientes)
- JavaScript (Fetch API)
- API: [ip-api.com](http://ip-api.com)

## Funcionalidades

- Mostra endereço IP público
- Localização (cidade, região, país)
- Provedor de internet (ISP)
- Fuso horário
- Design responsivo e moderno
